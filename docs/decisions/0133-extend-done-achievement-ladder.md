# 0133: Extend the Done Achievement Ladder

## Status

Accepted

## Date

2026-06-02

## Context

The first general achievement set gave Done a short badge ladder. Users who quickly earn every Done badge need visible next targets so the Achievements section continues to encourage steady work instead of feeling finished too early.

## Decision

Done achievements include a longer progression across total completions, high-output days, streaks, and full-week consistency. The ladder includes nearby continuation targets, such as 250 total done and a 30-day done streak, as well as longer-term targets like 500 and 1,000 total done and a 100-day done streak.

These badges remain derived presentation state from completed `RoutineLog` history.

## Consequences

- Users who have earned the initial Done badges still see meaningful in-progress targets.
- Done progress recognizes both volume and consistency rather than only a first completion, one century milestone, and one streak.
- Future Done badge additions should preserve a mix of near, medium, and long-term targets.
